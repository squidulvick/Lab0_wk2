Detailed Description

This program functions as the GUI for the measuring the time response in our circuit.

The step_response function is inserted into main on the micro controller and prompted to run from the GUI. This program then sets up the serial port so that the computer can read the data from the microcontroller and plots it. It is also then set up to compare the experimental data to a theoretical curve of the time response.
